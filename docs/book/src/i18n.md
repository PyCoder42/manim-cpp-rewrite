# Internationalization

Locale catalog continuity is maintained in `docs/i18n` for:

- `fr`
- `hi`
- `pt`
- `sv`

## Templates

The C++ docs pipeline maintains deterministic gettext templates at:

- `docs/i18n/gettext/book.pot`
- `docs/i18n/gettext/api.pot`

Use `./tools/i18n/extract_gettext.sh` to regenerate templates and
`./tools/docs/check_i18n_catalogs.sh` to validate locale catalog presence.
