# Internationalization

Locale catalog continuity is maintained in `docs/i18n` for:

- `fr`
- `hi`
- `pt`
- `sv`

## Templates

The C++ docs pipeline maintains deterministic gettext templates at:

- `docs/i18n/gettext/book.pot`
- `docs/i18n/gettext/api.pot`

Each supported locale stores translated catalogs scoped to the active docs
components:

- `docs/i18n/<locale>/LC_MESSAGES/book.po`
- `docs/i18n/<locale>/LC_MESSAGES/api.po`

Use `./tools/i18n/extract_gettext.sh` to regenerate templates,
`./tools/docs/check_i18n_catalogs.sh` to validate locale catalog presence, and
`./tools/docs/check_i18n_catalog_scope.sh` to enforce the scoped catalog set.
